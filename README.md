## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed
- A code editor (VS Code recommended)

### Run Day 1 Examples

```bash
cd firstDay
node variables.js
```

### Run Day 2 Examples

```bash
cd secondDay
npm install
node index.js
```
